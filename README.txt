== Description ==

Ever wanted to jump between pages & posts without using the sidebar? Now you can with a simple drop down list of ALL you pages and posts.
These links direct straight to the WYSIWYG editor for any give page/post.

== Installation ==

1. Upload the folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Done.
